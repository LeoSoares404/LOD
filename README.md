# LOD

Um **ARPG top-down pixel-art (16-bit)** — mistura de **Diablo** (loot, hordas, fantasia sombria) com **League of Legends** (habilidades com cooldown, orbes de vida/mana, combate baseado em skills).

Feito em **Godot 4**.

## 📄 Documentação

- **[GAME_DESIGN.md](GAME_DESIGN.md)** — plano de desenvolvimento completo, arquitetura e milestones.
- **[INSTALL.md](INSTALL.md)** — ambiente de desenvolvimento: tudo que precisa instalar.

## 🚀 Como rodar (após scaffold)

1. Instale o Godot 4 (ver [INSTALL.md](INSTALL.md)).
2. Abra o Godot → *Import* → selecione `project.godot`.
3. Pressione **F5** para rodar.
